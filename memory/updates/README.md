# Constitution & Spec Updates

This directory contains detailed summaries of constitution and spec updates, organized by date and version.

## Naming Convention

```
YYYY-MM-DD-vX.Y-brief-description.md
```

**Examples:**
- `2025-11-06-v1.1-gitops-rbac.md` - Added GitOps management and RBAC extensibility
- `2025-12-15-v1.2-observability.md` - Enhanced observability features
